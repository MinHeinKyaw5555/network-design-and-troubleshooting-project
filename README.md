# Network Design and Troubleshooting Project

## Overview

This project focused on designing and troubleshooting a Local Area Network (LAN) for a corporate office environment.

## Project Objectives

- Design network infrastructure for business requirements
- Configure network devices and services
- Apply TCP/IP networking concepts
- Implement DNS and DHCP services
- Perform IPv4 addressing and subnetting
- Troubleshoot network connectivity issues

- Subnetting
